# Crop-Farming-Game
Language: C++
Description:
Object-oriented final project for a computer science class using Visual Studio.  
User interactive game using pointers, objects, and basic game loops.  
Implemented multiple libraries, classes, and features to save and load previously saved files.

How to play:
It is a simple game where you have the option to grow a crop and help it survive 10 days to win. 
Everyday you must make a decision towards the crop. 
You can water (w), fertilize (f), treat bug infestations (b), and treat fungus infestations (t).
As days go by if the crop has more or less water/fertilizer than it should health will decrease.
Similarly, any infestation will greatly decrease the crop's health.
You lose if the crops health drops bellow zero before the 10 days.
You can always quit, save your file, and load it later. 
Good luck!
